## Architecture decisions

1. **GitHub Actions over Vercel native** — quality gate (tsc + lint + build must pass before deploy), scheduled jobs, free for public repos
2. **Daily watchlist + weekly full** — balance data freshness vs Yahoo rate limits
3. **MIT license** — most permissive, privacy-preserving (GitHub handle, not real name)
4. **Grouped Dependabot** — patch+minor in one PR, major individual — less noise
